CSS · Урок 03

Блокова Модель

Кожен елемент — прямокутний блок. Width, height, border, padding, margin, overflow, visibility
Теорія Предмет: Веб Клас: 7 ~40 хв
01

Що таке блокова модель?

Усі елементи на веб-сторінці браузер розглядає як прямокутні блоки (box). Блокова модель визначає чотири шари кожного елемента:

┌─────────────────────────────────┐ │ MARGIN │ │ ┌───────────────────────────┐ │ │ │ BORDER │ │ │ │ ┌─────────────────────┐ │ │ │ │ │ PADDING │ │ │ │ │ │ ┌───────────────┐ │ │ │ │ │ │ │ CONTENT │ │ │ │ │ │ │ │ width×height │ │ │ │ │ │ │ └───────────────┘ │ │ │ │ │ └─────────────────────┘ │ │ │ └───────────────────────────┘ │ └─────────────────────────────────┘
  • width / height — розмір області вмісту
  • padding — простір між вмістом і рамкою
  • border — рамка навколо елемента
  • margin — простір зовні рамки
02

Висота та ширина (height, width)

За замовчуванням браузер підлаштовує розмір елемента під вміст. CSS дозволяє задати точні розміри:

p {
  height: 80px;
  width:  240px;
}
⚠ УВАГА
Фіксовані пікселі дають точний розмір, але можуть зламати макет на мобільних пристроях. Для адаптивного дизайну використовуйте %, min-width, max-width.
03

Рамка (border)

Border — лінія навколо елемента. Три обов'язкові параметри: товщина, стиль, колір:

p {
  border: 3px solid coral;
}

Стилі рамки: solid, dashed, dotted, none.

За замовчуванням: border: medium none currentColor — рамки не видно.

04

Закруглення рамки (border-radius)

Закруглює кути елемента. Значення — радіус кривизни:

div {
  border:        3px solid blue;
  border-radius: 8px;    /* закруглені кути */
}

.avatar {
  width:         60px;
  height:        60px;
  border-radius: 50%;   /* ідеальне коло */
}
💡 ПІДКАЗКА
border-radius: 50% створює коло лише якщо width === height.
05

Внутрішній відступ (padding)

Padding — простір між вмістом і рамкою. Він використовує колір фону елемента:

p {
  padding: 10px;           /* з усіх боків */
  padding: 10px 20px;     /* top/bottom  left/right */
  padding: 5px 10px 15px; /* top  left/right  bottom */
}

/* Або окремо: */
p {
  padding-top:    10px;
  padding-right:  20px;
  padding-bottom: 10px;
  padding-left:   20px;
}
06

Зовнішній відступ (margin)

Margin — простір зовні елемента, між ним і сусідніми елементами. Скорочений запис такий самий, як у padding:

p {
  margin: 20px;         /* з усіх боків */
  margin: 0 auto;      /* центрування по горизонталі */
}

Злиття margin: вертикальні відступи двох сусідніх елементів зливаються — береться більший. Горизонтальні завжди додаються.

💡 ЦЕНТРУВАННЯ
margin: 0 auto центрує блок горизонтально. Обов'язкова умова — елемент повинен мати задану ширину (width).
07

Мінімальна та максимальна ширина/висота

Задають межі розміру елемента для адаптивного дизайну:

p {
  min-width:  300px; /* не вужче */
  max-width:  600px; /* не ширше */
  min-height: 150px; /* не нижче */
  max-height: 300px; /* не вище */
}
⚠ УВАГА
Якщо max-height задано занадто малим, вміст може «вилізти» за межі елемента. Вирішення — властивість overflow.
08

Переповнення (overflow)

Якщо вміст більший за контейнер, overflow визначає що з ним робити:

div { overflow: visible; } /* за замовчуванням — виходить за межі */
div { overflow: hidden; }  /* обрізається */
div { overflow: scroll; }  /* прокрутка */

Для керування окремими осями: overflow-x (горизонталь) та overflow-y (вертикаль).

09

Скидання стандартних стилів (normalize)

Браузери додають власні margin і padding до елементів, а ширина елементів рахується по-різному залежно від браузера. Розробники скидають це на самому початку CSS:

* {
  margin:      0;
  padding:     0;
  box-sizing: border-box;
}
💡 ПІДКАЗКА
Значення 0 не потребує одиниці виміру. Явно написані правила (margin-top: 24px) перевизначать скидання завдяки вищій специфічності. Цей блок завжди пишеться першим рядком у файлі CSS.
10

box-sizing: border-box

За замовчуванням (content-box) властивості width і height задають розмір лише вмісту. Padding і border додаються зверху — реальний елемент виявляється ширшим, ніж написано в CSS:

content-box (DEFAULT) ───────────────────────────────────── width: 200px → лише контент реальна ширина = 200 + 20 + 20 + 2 + 2 = 244px ↑padding↑ ↑border↑
.box {
  width:      200px;
  padding:    20px;
  border:     2px solid black;
  /* реальна ширина = 200 + 20×2 + 2×2 = 244px */
}

З box-sizing: border-box браузер вписує padding і border всередину заданого розміру. Що написано в CSS — те й займає місце на сторінці:

border-box ───────────────────────────────────── width: 200px → разом із padding і border реальна ширина = 200px ✓
.box {
  width:       200px;
  padding:     20px;
  border:      2px solid black;
  box-sizing: border-box;
  /* реальна ширина = 200px */
}
✓ НАЙКРАЩА ПРАКТИКА
Завжди починай CSS з нормалізації: * { margin: 0; padding: 0; box-sizing: border-box; }. Це усуває браузерні відступи і робить розрахунок розмірів передбачуваним.

11

Видимість (visibility)

Приховує або показує елемент:

.hidden { visibility: hidden; } /* невидимий, але займає місце */
.gone   { display:     none; }   /* повністю видалений з макету */
