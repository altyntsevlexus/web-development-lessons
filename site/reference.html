Урок 01

Селектори

Селектор — перша частина CSS-правила. Він вказує до яких HTML-елементів застосовувати стилі. Від правильного вибору залежить, чи зміниться один елемент чи одразу сотня.

element
Вибирає всі елементи з таким тегом на сторінці.
p  { color: red; }
h1 { font-size: 32px; }
Увага: змінює всі <p> на сторінці. Використовуй для базових налаштувань.
.class
Вибирає всі елементи з атрибутом class="...". Один клас можна давати багатьом елементам.
.card   { background: white; }
.active { color: green; }
Найчастіший вибір у реальних проектах. Один елемент може мати кілька класів: class="card active"
#id
Вибирає один конкретний елемент за унікальним id. ID на сторінці не повторюється.
#header { height: 80px; }
#logo   { width: 120px; }
Специфічність #id дуже висока — важко перевизначити. Надавай перевагу класам.
*
Універсальний — вибирає всі елементи на сторінці без винятку.
* {
  margin:     0;
  padding:    0;
  box-sizing: border-box;
}
Майже єдине практичне застосування — нормалізація на початку CSS-файлу.
A B  (нащадок)
Вибирає всі елементи B, що знаходяться всередині A (на будь-якому рівні вкладеності).
.menu a   { color: white; }
.card p   { font-size: 14px; }
A > B  (дитина)
Вибирає B лише якщо B — прямий нащадок A (не вкладений глибше).
ul > li { list-style: none; }
/* лише <li> одразу в <ul> */
:hover
Застосовується коли курсор знаходиться над елементом.
a:hover       { color: blue; }
.btn:hover    { opacity: .8; }
.card:hover   { transform: translateY(-4px); }
:first-child
:last-child
:nth-child()
Вибирає елементи за позицією серед «братів».
li:first-child    { font-weight: bold; }
li:last-child     { border: none; }
tr:nth-child(2n)  { background: #f5f5f5; }
/* 2n = кожен парний рядок */

Урок 02

Текст і шрифт

Властивості для керування виглядом тексту — шрифт, розмір, колір, вирівнювання, відстань між літерами та рядками.

color
Колір тексту. Приймає назви, hex, rgb, hsl.
color: red;
color: #ff4444;
color: rgb(255, 68, 68);
color: rgba(255, 68, 68, 0.8); /* + прозорість */
font-family
Шрифт. Завжди вказуй запасний на випадок, якщо основний не знайдено.
font-family: 'Roboto', sans-serif;
font-family: Georgia, serif;
font-family: 'Courier New', monospace;
Google Fonts: підключай у <head> через <link>, потім використовуй в CSS.
font-size
Розмір шрифту. Найчастіше px або rem.
font-size: 16px;   /* пікселі */
font-size: 1rem;   /* відносно кореня (16px) */
font-size: 1.25em; /* відносно батьківського */
font-size: 2vw;    /* % від ширини вікна */
font-weight
Жирність тексту.
font-weight: normal;  /* = 400 */
font-weight: bold;    /* = 700 */
font-weight: 300;     /* тонкий */
font-weight: 900;     /* найжирніший */
Числові значення доступні лише якщо шрифт має відповідні накреслення.
text-align
Горизонтальне вирівнювання тексту всередині блокового елемента.
text-align: left;    /* за замовчуванням */
text-align: center;
text-align: right;
text-align: justify; /* по ширині */
text-decoration
Декоративна лінія у тексті.
text-decoration: none;        /* прибрати підкреслення у <a> */
text-decoration: underline;   /* підкреслення */
text-decoration: line-through;/* закреслення */
text-transform
Регістр тексту — без зміни HTML.
text-transform: uppercase;   /* ВЕЛИКІ */
text-transform: lowercase;   /* маленькі */
text-transform: capitalize;  /* Перша Велика */
line-height
Висота рядка — відстань між рядками тексту.
line-height: 1;     /* щільно */
line-height: 1.5;   /* комфортно для тексту */
line-height: 24px;  /* точне значення */
Без одиниці виміру (наприклад 1.5) — множник від поточного font-size. Це краще.
letter-spacing
Відстань між літерами.
letter-spacing: 0;     /* за замовчуванням */
letter-spacing: 2px;   /* розріджено */
letter-spacing: -1px;  /* щільніше */
Позитивне значення — розріджує (часто для заголовків uppercase). Негативне — стискає.
font-style
Курсив або нормальне написання.
font-style: normal;  /* за замовчуванням */
font-style: italic;  /* курсив */

Урок 02

Фон

Властивості, що керують фоном елемента — колір, зображення, розмір і позиція фонового зображення.

background-color
Колір фону елемента.
background-color: white;
background-color: #1a1a2e;
background-color: rgba(0, 0, 0, 0.5); /* напівпрозорий */
background-color: transparent;         /* без фону */
background-image
Фонове зображення або градієнт.
background-image: url('photo.jpg');
background-image: linear-gradient(
  to bottom,
  #1a1a2e,
  #16213e
);
background-size
Розмір фонового зображення.
background-size: cover;    /* заповнює, може обрізати */
background-size: contain;  /* вміщується повністю */
background-size: 100% 100%;/* точний розмір */
cover — найчастіший вибір для банерів і hero-секцій.
background-position
Початкова позиція фонового зображення.
background-position: center;
background-position: top left;
background-position: 50% 30%;
Разом з cover і center — фото завжди по центру, скільки б не змінювався розмір вікна.
opacity
Прозорість всього елемента разом із вмістом. Від 0 (невидимий) до 1 (повністю непрозорий).
opacity: 1;    /* за замовчуванням */
opacity: 0.5;  /* 50% прозорий */
opacity: 0;    /* повністю прозорий */
Відмінність від rgba: opacity робить прозорим і текст усередині. rgba() — лише фон.
background-repeat
Чи повторюється фонове зображення.
background-repeat: no-repeat; /* не повторювати */
background-repeat: repeat;    /* за замовчуванням */
background-repeat: repeat-x;  /* лише горизонталь */

Урок 03

Блокова модель

Кожен елемент — прямокутний блок із чотирьох шарів: content → padding → border → margin. Розуміння цих шарів — основа верстки.

Блокова модель — як це виглядає
margin → border → padding → content
margin
border + padding
content · width × height
box-sizing
Визначає чи включається padding і border у задану ширину.
/* Завжди пиши першим рядком */
* { box-sizing: border-box; }

/* content-box (default):
   width: 200px + padding 20px + border 2px
   → реальна ширина: 244px ⚠ */

/* border-box:
   width: 200px (все включено)
   → реальна ширина: 200px ✓ */
width / height
Розмір блоку. За замовчуванням — розмір вмісту (з border-box — увесь елемент).
width:  300px;
height: 200px;
width:  50%;   /* % від батька */
width:  100vw; /* % від вікна */
min / max width · height
Межі розміру — для адаптивного дизайну.
min-width:  200px; /* не вужче */
max-width:  800px; /* не ширше */
min-height: 100px;
max-height: 400px;
max-width: 860px; margin: 0 auto — класичний контейнер сторінки.
padding
Внутрішній відступ — простір між вмістом і рамкою. Успадковує колір фону.
padding: 20px;          /* всі 4 боки */
padding: 10px 20px;     /* top/bottom  left/right */
padding: 5px 10px 15px; /* top  lr  bottom */
padding-top:    10px;   /* окремо */
padding-right:  20px;
padding-bottom: 10px;
padding-left:   20px;
margin
Зовнішній відступ — простір між елементами. Синтаксис як у padding.
margin: 20px;      /* всі 4 боки */
margin: 0 auto;   /* горизонтальне центрування */
margin-top: 40px; /* окремо */
Злиття margin: вертикальні відступи двох сусідніх елементів зливаються — береться більший.
border
Рамка навколо елемента. Три параметри: товщина, стиль, колір.
border: 1px solid #ccc;
border: 2px dashed coral;
border: 3px dotted blue;
border-bottom: 2px solid black; /* лише знизу */
border: none; /* прибрати */
border-radius
Закруглення кутів.
border-radius: 4px;   /* легке закруглення */
border-radius: 12px;  /* сучасні картки */
border-radius: 50%;   /* коло (якщо width=height) */
border-radius: 999px; /* таблетка */
overflow
Що робити якщо вміст виходить за межі елемента.
overflow: visible; /* виходить (за замовчуванням) */
overflow: hidden;  /* обрізається */
overflow: scroll;  /* прокрутка завжди */
overflow: auto;    /* прокрутка коли потрібно */
overflow-y: hidden; /* лише вертикаль */
visibility
Показує або приховує елемент зберігаючи його місце.
visibility: visible; /* за замовчуванням */
visibility: hidden;  /* прихований, але місце є */
Порівняй: display: none — видалений з макету, місця немає. visibility: hidden — місце залишається.

Урок 04

Display

display визначає як елемент поводиться в потоці документа — займає весь рядок, стоїть в рядку з іншими, або взагалі зникає.

Порівняння трьох значень
display: block — кожен займає весь рядок
block елемент
block елемент
display: inline — течуть у тексті, без width/height
inline inline inline
display: inline-block — рядок + можна задати розміри
inline-block inline-block inline-block
display: block
Новий рядок, повна ширина батька. Підтримує width, height, всі margin і padding.
/* Block за замовчуванням: */
div, p, h1–h6, section,
article, ul, ol, li

/* Зробити block: */
span { display: block; }
display: inline
Не починає новий рядок. width і height не діють. Вертикальні margin теж ігноруються.
/* Inline за замовчуванням: */
span, a, strong, em, img

/* Зробити inline: */
li { display: inline; }
/* → горизонтальне меню */
display: inline-block
Стоїть в рядку як inline, але підтримує width, height і всі відступи як block.
.btn {
  display:  inline-block;
  width:    140px;
  height:   44px;
  padding:  10px 24px;
}
Часто використовують для кнопок, чіпів, іконок з текстом. Сучасна альтернатива — flex.
display: none
Елемент повністю зникає з макету — місця не займає, не клікабельний.
.hidden { display: none; }

/* Показати при hover на батьку: */
.card:hover .tooltip {
  display: block;
}
Порівняй з visibility: hidden — те ж приховування, але місце залишається.

Урок 04

Position

position дозволяє вийти з нормального потоку і точно розмістити елемент за координатами. Властивості top, right, bottom, left задають зсув.

position: static
Значення за замовчуванням. Елемент у нормальному потоці. top/left/z-index не діють.
div { position: static; }
/* писати не потрібно — це default */
position: relative
Залишається в потоці, але може зміщуватись від свого місця. Головне — стає контейнером для absolute-нащадків.
.parent {
  position: relative; /* контейнер */
}
.shifted {
  position: relative;
  top: 10px; left: 20px;
}
position: absolute
Виходить з потоку. Позиціонується відносно найближчого positioned-предка (не static).
.card   { position: relative; }

.badge  {
  position: absolute;
  top:   12px;
  right: 12px;
}
.card — position: relative
.badge
position: fixed
Фіксується відносно вікна браузера. Не рухається при прокрутці.
.navbar {
  position: fixed;
  top:      0;
  left:     0;
  width:    100%;
  z-index:  100;
}
Виходить з потоку — додай padding-top до body рівний висоті navbar.
position: sticky
Гібрид: веде себе як relative, але «прилипає» до краю при прокрутці.
.section-header {
  position: sticky;
  top:      0; /* поріг прилипання */
  z-index:  10;
}
Не працює якщо батьківський елемент має overflow: hidden або overflow: auto.
z-index
Порядок шарів коли елементи перекриваються. Більше — вище. Працює лише з positioned-елементами.
.content { z-index: 1;    }
.header  { z-index: 100;  }
.modal   { z-index: 1000; }
.tooltip { z-index: 9999; }
Умова: елемент повинен мати position, відмінний від static.
top · right · bottom · left
Координати зсуву для relative, absolute, fixed, sticky.
/* absolute: прилипити до кутів */
top: 0; left: 0;   /* верхній лівий */
top: 0; right: 0;  /* верхній правий */

/* по центру (absolute) */
top: 50%; left: 50%;
transform: translate(-50%, -50%);
inset
Скорочення для одночасного задання top + right + bottom + left.
inset: 0;         /* = top/right/bottom/left: 0 */
inset: 10px 20px; /* top/bottom: 10  left/right: 20 */
Зручно для position: fixed; inset: 0 — розтягує на весь екран (overlay).
