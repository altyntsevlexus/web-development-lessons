CSS · Урок 02

Візуальні Правила

Шрифт, розмір, вирівнювання, колір, прозорість і фонові зображення
Теорія Предмет: Веб Клас: 7 ~25 хв
01

Шрифт (font-family)

Властивість font-family задає шрифт тексту. Якщо назва шрифту складається з кількох слів — вона береться в лапки:

h1 {
  font-family: Garamond;
}

h2 {
  font-family: "Courier New"; /* кілька слів — у лапках */
}
💡 ПІДКАЗКА
Шрифт повинен бути встановлений на комп'ютері користувача. Стандартний шрифт браузера — Times New Roman. Для веб-шрифтів використовуйте Google Fonts.
02

Розмір шрифту (font-size)

Властивість font-size задає розмір тексту. Найпоширеніша одиниця — пікселі (px):

p {
  font-size: 18px;
}

Також можна використовувати em (відносно батьківського елемента), rem (відносно кореневого елемента), %.

03

Жирність шрифту (font-weight)

Властивість font-weight задає товщину (жирність) шрифту:

p    { font-weight: bold; }   /* жирний */
span { font-weight: normal; } /* звичайний */

Замість ключових слів можна використовувати числа: 100 (найтонший) … 900 (найжирніший). Стандартне значення — 400, жирне — 700.

04

Вирівнювання тексту (text-align)

Властивість text-align керує горизонтальним вирівнюванням тексту всередині елемента:

h1 { text-align: center; } /* по центру */
p  { text-align: left; }   /* зліва (за замовчуванням) */
.q { text-align: right; }  /* справа */
05

Колір (color та background-color)

color — колір тексту. background-color — колір фону елемента:

h1 {
  color: red;             /* колір тексту */
  background-color: blue; /* колір фону */
}

Колір можна задавати різними способами:

  • Назва: red, coral, teal
  • HEX: #ff0000, #00d4aa
  • RGB: rgb(255, 0, 0)
  • RGBA (з прозорістю): rgba(0, 212, 170, 0.5)
06

Прозорість (opacity)

Властивість opacity задає прозорість всього елемента. Значення від 0 (повністю прозорий) до 1 (повністю непрозорий):

.overlay {
  opacity: 0.5; /* 50% прозорості */
}

.hidden {
  opacity: 0;   /* невидимий, але займає місце */
}
💡 ПІДКАЗКА
opacity впливає на весь елемент разом із вмістом. Для прозорості лише фону використовуйте rgba() у background-color.
07

Фонове зображення (background-image)

Властивість background-image задає зображення як фон елемента:

.hero {
  background-image: url("https://example.com/photo.jpg");
}

Зазвичай разом із зображенням використовують додаткові властивості:

.hero {
  background-image:    url("photo.jpg");
  background-size:     cover;   /* заповнює весь елемент */
  background-position: center;  /* центрує зображення */
  background-repeat:   no-repeat;
}
