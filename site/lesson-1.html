CSS Селектори

Як підключити CSS до HTML і як вибирати елементи для стилізації
Теорія ~35 хв
01

Що таке CSS?

CSS (Cascading Style Sheets — Каскадні таблиці стилів) — це мова, яку веб-розробники використовують для стилізації HTML-контенту на веб-сторінці.

Кожне CSS-правило складається з чотирьох частин:

h1 {
  color: blue;
}
  • Селектор — h1 (який елемент стилізуємо)
  • Властивість — color (що змінюємо)
  • Значення — blue (яке значення задаємо)
  • Декларація — color: blue; (властивість + значення разом)
02

Вбудовані стилі (Inline)

Стиль можна додати прямо в HTML-тег через атрибут style:

<p style="color: red; font-size: 20px;">Текст</p>
⚠ АНТИПАТЕРН
Вбудовані стилі важко підтримувати: щоб змінити один колір, доведеться правити кожен тег окремо. Використовуйте лише у крайніх випадках.
03

Тег <style> (внутрішня таблиця)

CSS можна писати між тегами <style> всередині <head>:

<head>
  <style>
    p { color: red; font-size: 20px; }
  </style>
</head>
⚠ АНТИПАТЕРН
Внутрішня таблиця підходить лише для маленьких сторінок. У реальних проєктах CSS зберігають окремо.
04

Зовнішній CSS-файл ✅

Найкращий підхід — окремий файл .css, підключений через тег <link> у <head>:

<head>
  <link href="style.css" type="text/css" rel="stylesheet">
</head>

Тег <link> потребує трьох атрибутів: href (шлях до файлу), type (завжди text/css), rel (завжди stylesheet).

✓ НАЙКРАЩИЙ ПІДХІД
Один CSS-файл керує стилями всього сайту. Зміна в одному місці — оновлення на всіх сторінках.
05

Селектор за тегом і універсальний *

Селектор за тегом стилізує всі елементи з вказаним тегом:

p  { color: green; }
h1 { color: maroon; }

Універсальний селектор * вибирає всі елементи на сторінці. Зазвичай використовується для скидання стилів:

* {
  margin: 0;
  padding: 0;
}
💡 ПІДКАЗКА
* має найнижчу специфічність (0) — будь-який інший селектор перевизначить його.
06

Селектор за класом

Додайте атрибут class до HTML-елемента, а в CSS використовуйте крапку перед назвою класу:

/* HTML */
<p class="title">Заголовок</p>

/* CSS */
.title { color: teal; }

Елемент може мати кілька класів (через пробіл у HTML):

<h1 class="title uppercase">Заголовок</h1>

.uppercase { text-transform: uppercase; }
07

Селектор за ID та специфічність

ID — унікальний ідентифікатор елемента. У CSS використовується #:

<h1 id="article-title">Великий заголовок</h1>

#article-title { font-family: cursive; }

Специфічність — порядок пріоритету (від найвищого до найнижчого):

h1        { color: red; }        /* тег    → 001 */
.headline { color: firebrick; } /* клас   → 010 */
#heading  { color: blue; }       /* id     → 100 */
/* Переможе: blue */
💡 ПРАВИЛО
ID → Клас → Тег. Використовуйте ID рідко — тільки для дійсно унікальних елементів.
08

Селектор за атрибутом і псевдокласи

Атрибутний селектор стилізує елементи за наявністю або значенням атрибута:

[disabled]        { opacity: 0.5; }
[type="text"]     { border: 1px solid gray; }
[href^="https"]   { color: green; }  /* починається з */
[href$=".pdf"]    { color: red; }    /* закінчується на */

Псевдоклас описує стан або позицію елемента (через :):

a:hover       { color: darkorange; }
li:first-child{ font-weight: bold; }
input:focus   { border-color: blue; }
09

Ланцюжок, вкладеність, групування

Ланцюжок — вимагає декількох селекторів одночасно:

h1.special { color: gold; } /* тільки h1 з класом "special" */

Вкладеність (нащадок) — вибирає елемент всередині іншого:

.main-list li { color: green; } /* li всередині .main-list */

Групування — кілька селекторів через кому:

h1,
.menu { font-family: Georgia; }

!important — перевизначає будь-який стиль незалежно від специфічності:

p { color: blue !important; }
⛔ УНИКАЙТЕ
!important ускладнює підтримку коду. Використовуйте лише як крайній захід.
